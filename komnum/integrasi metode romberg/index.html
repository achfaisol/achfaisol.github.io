    Penyelesaian Integral Menggunakan Metode Romberg¶

    Metode Romberg¶

    Metode integrasi Romberg didasarkan pada perlusaan ekstrapolasi Richardson untuk memperoleh nilai integrasi yang semakin baik. Sebagai catatan, setiap penerapan ekstrapolasi Richardson akan menaikkan order galat pada hasil solusinya sebesar dua.

    Implementasi Metode Romberg dengan Python¶

    import numpy as np

def trapezcomp(f, a, b, n):
    h = (b - a) / n
    x = a
    In = f(a)
    for k in range(1, n):
        x = x + h
        In += 2*f(x)
    return (In + f(b))*h*0.5

def romberg(f, a, b, n):
    R = np.zeros((n, n))
    print('Tabel Romberg')
    for k in range(0, n):
        R[k, 0] = trapezcomp(f, a, b, 2**k)
        for j in range(0, k):
            R[k, j+1] = (4**(j+1) * R[k, j] - R[k-1, j]) / (4**(j+1) - 1)

        print(R[k, 0:k+1])

    return R
